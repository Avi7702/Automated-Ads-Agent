## WHAT THIS APP DOES

Users upload boring product photos. They describe what they want in plain English. The app understands their intent dynamically and generates professional marketing visuals.

## SMART PRODUCT RESEARCH (CONDITIONAL)

Research is triggered ONLY when needed:

**DO research when:**
- Product type is unclear from image
- User asks about installation/combination (need to know HOW they work together)
- Industry-specific context needed
- User explicitly asks for accurate representation

**SKIP research when:**
- Simple lifestyle/background swap
- User provides clear context
- Speed is priority
- Seasonal/style changes only

### Research Query Structure

When research IS needed, use Google Search grounding to find:

```
For product identification:
â†’ "What is [visual description of product]"
â†’ "What is [product] used for"

For combination/installation:
â†’ "How to install [product A] with [product B]"
â†’ "How do [product A] and [product B] work together"

For industry context:
â†’ "[Product type] professional photography"
â†’ "[Product] in use on [industry] site"

For aspirational context:
â†’ "Premium [product category] marketing photography"
â†’ "Lifestyle [product] photography examples"
```

## PROMPT CONSTRUCTION ENGINE

Based on detected intent, build prompt dynamically:

### Base Structure

```
[SCENE PLACEMENT] + [ENVIRONMENT DETAILS] + [LIGHTING/MOOD] + 
[PRODUCT FOCUS] + [STYLE INSTRUCTIONS] + [TECHNICAL SPECS]
```

### Template by Content Type

**LIFESTYLE TRANSFORMATION:**
```
"Place this [product] naturally in [environment]. [Time of day] lighting, 
[mood] atmosphere. The product is clearly visible but feels organic to 
the scene. [Style] photography, [platform-optimized] composition."
```

**INSTALLATION/COMBINATION:**
```
"Show [product A] and [product B] [installed/connected/working together] 
in a [professional/realistic] [industry] setting. The connection/relationship 
between products is clear. [Lighting], authentic feel, demonstrates proper use."
```

**BEFORE/AFTER:**
```
"Create a [split/comparison] composition showing: Left side - [problem state]. 
Right side - [solution using product]. Clear visual contrast, same angle/lighting 
for comparison. The product's value is immediately obvious."
```

**SCALE DEMONSTRATION:**
```
"Show this [product] with [human figure/common object] for scale reference. 
The size relationship is immediately clear. [Professional/lifestyle] setting, 
the product's scale is the focus while maintaining [mood] feel."
```

**SEASONAL STYLING:**
```
"Style this [product] for [season/holiday]. Add [seasonal elements] while 
keeping product as hero. [Warm/cool/festive] color palette, [mood] atmosphere. 
The product remains professional, the seasonal theme enhances not overwhelms."
```

**PROBLEM/SOLUTION:**
```
"Visual showing the problem of [issue] alongside the solution using [product]. 
[Before state] transforms to [after state]. The product's value proposition 
is immediately clear. Professional, compelling composition."
```

**PRODUCT FAMILY:**
```
"Arrange [these products] together as a cohesive collection/family. 
[Organized/artistic] layout showing how they work as a system. Consistent 
lighting across all products, [premium/professional] product photography style."
```

**ACTION/IN-USE:**
```
"Show [product] being actively used by [appropriate person] in [context]. 
Capture the moment of use, [professional/authentic] feel. The product's 
purpose is immediately clear, [dynamic/documentary] composition."
```

**INDUSTRY CONTEXT:**
```
"Place [product] in an authentic [industry] professional setting. 
[Workers/context] in background, the product is hero but environment 
is realistic. [Time of day] lighting, [documentary/professional] style, 
aspirational but believable."
```

## CAPTION & HASHTAG GENERATION

After image generation, automatically create:

### Caption Generation Rules
- Match the detected mood/platform
- Keep it concise (under 150 characters for Twitter, flexible for others)
- Include subtle call-to-action when appropriate
- Professional tone for LinkedIn, casual for Instagram
- Never cringey or over-promotional

### Hashtag Generation Rules
- 5-8 relevant hashtags
- Mix of: industry, product type, mood, platform-trending
- Avoid overused/spammy tags
- Match the professional level of the content

## SESSION MANAGEMENT

### Per Session Limits
```
- Track generation count per session
- After 15 generations: "You've created 15 images! Want to continue?"
- After 20 generations: "Consider downloading your favorites - 
                         session will reset soon"
- Store generated images temporarily (1 hour)
- Provide "Download All" option
```

### Image Storage
```
- Save generated images to temp storage
- Provide download links (1K standard, 2K, 4K options)
- Clear after session ends or 1 hour timeout
- No permanent storage (user downloads what they want)
```
