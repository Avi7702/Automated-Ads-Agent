## Before You Finish - Quality Checklist

Before completing this task, please verify:

**Testing:**
- [ ] Did you run the tests? (`npm test`, `pytest`, etc.)
- [ ] Do all tests pass?
- [ ] Did you add tests for new functionality?

**Code Quality:**
- [ ] Does the code follow project conventions?
- [ ] Are there any linting errors?
- [ ] Did you remove debug code (console.log, print, etc.)?

**Build:**
- [ ] Does the project build successfully?
- [ ] Are there any TypeScript/compilation errors?

If you haven't done these, consider running them now before finishing.
